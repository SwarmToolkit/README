# README
Not Affiliate of Turtle Beach.. 
SwarmToolkit is an independent, open-source editor for Turtle Beach Swarm II keyboard profiles. It allows users to create, edit, preview, and share lighting themes while exporting compatible profile files for use with Swarm II.
Right now it will only support Vulkan TKL / Open to contributions if you have another Turtle Beach Keyboard.
Goals 
To create a better version of Swarm with more features and easier accessibility. 
Easily export created profiles
